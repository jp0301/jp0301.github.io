## 자바란?

**자바의 탄생**

- 1995년에 썬 마이크로시스템즈에서 발표한 객체지향 언어
- 현재는 썬 마이크로시스템즈는 오라클에 인수 된 상태

**자바의 특징**

- 쉬운 언어이다.

  - C와 C++ 언어의 문법을 기본으로 차용하여 개발됨
  - C와 C++ 이 가진 어려운 문법인 포인터와 다중 상속 제거
  - C와 C++ 에 비해 쉬운 언어

- 플랫폼에 독립적이다.

  - 자바는 JVM(Java Virtual Machine)만 있으면 윈도우, 리눅스, 맥 등 어떤 플랫폼에서도 실행 가능
  - `JVM - 자바로 만들어진 프로그램을 실행해주는 프로그램`

- 객체지향 언어이다.
- Garbage Collector 로 사용되지 않는 메모리 관리를 자동적으로 정리 해준다.

## 자바 개발순서

1. 코드 작성
2. 코드 컴파일
3. JVM으로 실행

<br />

**따라해보기**

- 소스 작성
  - HelloWorld.java 파일을 C:\tmp 디렉토리에 작성해보고
  - 자바는 파일 이름이 중요한데 대소문자 구별을 하기 때문에 주의!

```Java
  public class HelloWorld{
    public static void main(String[] args) {
      System.out.println("Hello World");
    }
  }
```

- 컴파일

  - 윈도우 콘솔(cmd 창 열어서 실행 가능)
  - 디렉토리로 이동해서 javac HelloWorld.java 로 컴파일

- 실행
  - java HelloWorld.java 로 실행
  - 화면에 결과 출력

`이클립스로 실행하는 방법은 생략`

## 주석문

주석이란, 프로그램의 코드와 실행에 영향을 주지 않는 문장

**주석의 종류**

- 구현 주석
  - 행단위 주석 ( //를 해주면, 해당 행이 주석 처리됨 )
  - 블럭단위 주석 ( /_ 내용 _/ )
- 문서화 주석

  - /\*\*
  - \* 문서 포함내용
  - \*/
  - 문서화 주석은 클래스, 인터페이스 그리고 멤버 당 하나씩 가질 수 있고, 선언 바로 전에 작성

<br />

`주석 단축키` : Ctrl + /
