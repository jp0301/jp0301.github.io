BlockChain MarketPlace 간단 구현
8개월 전

NFT(Non-Fungible Token)에 관한 아이디어로 요즘 MZ 세대는 멤버쉽, 과시욕, 브랜딩, 정체성 표현들의 다양한 이유로 NFT를 구매한다고도 합니다. 그들이 왜 NFT를 사고 그들이 보고 있는 가능성이 무엇인지에 대해 알아보는 척 하면서 간단하게 NFT를 사고팔 수 있는 마켓플레이스를 만들어보는 경험을 해보았습니다.

즉, 그냥 프로젝트를 통해 NFT 마켓 웹을 한번 구축해보자. NFT 생성(발행, 민팅), 구매, 판매를 구현해보자.

Dapp 구성

사용자가 디앱에 접속을 해서 사용자 컴퓨터에서 메타마스크 같은 개인지갑을 이용해서 블록체인에 접속 접속을 할 때 web3.js 라는 sdk(소프트웨어 개발 키트)를 사용한다.

블록체인에는 미리 만들어진 스마트 컨트랙트(계약서)가 배포가 되는 것이고 스마트 컨트랙트와 web3.js가 상호작용을 해서 블록체인의 데이터를 가져오고 사용하는 기능으로 흘러간다.

IPFS는 사용자가 NFT를 발행할 때 사용하는 이미지 컨텐츠, 설명 등을 다룬 메타데이터 파일을 IPFS에서 업로드하는 기능을 담당한다.

시연영상

소감

점차 대세로 확장되고 있는 블록체인 업계에 대한 관심을 가질 수 있는 계기였으며 스마트 컨트랙트와 WEB3를 연계해보면서 솔리디티 작성 문화와 프론트엔드 지식에 도움이 되었습니다.

프로젝트를 진행하면서 찾아보니 NFT 마켓플레이스를 빌더해주는 서비스도 있었습니다. 자신의 NFT 프로젝트를 빌더를 통해 자신만의 NFT 스토어를 열 수 있고 기존 마켓들과도 연동이 가능하게 해줬습니다. 비즈니스와 서비스 부분에서도 상당히 많은 것을 고려해야 되는구나 싶었습니다.
