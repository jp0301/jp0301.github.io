계속해서 업데이트하면서 알고리즘 공부 방향을 정립해서 꾸준히 해나가볼 예정이다.

1. 알고리즘 문제

  1. 코드업: 기초 100제 / 파이썬
  2. 백준 온라인 저지: 단계별 문제
  3. 프로그래머스: 레벨별 테스트
  4. LeetCode :
  5. 이것이 취업을 위한 코딩 테스트다 with 파이썬 (저자. 나동빈)

2. 알고리즘 공부 방법

  1. 자바와 파이썬 위주로 언어 공부
  2. 백준과 코드업에서 기초, 쉬운 문제 풀이
  3. 유형별 알고리즘 이론과 기출문제 풀이
  4. 이코테 책 완독 후 백준 온라인 저지에서 삼성 SW 역량테스트 문제 풀기
  5. 책 2부와 3부 중심으로 주요 알고리즘 유형 복습

요약

  • 언어 선택 및 공부
  • 백준, 코드업 쉬운 문제 공략
  • 프로그래머스 자료구조 공부
  • 백준 및 프로그래머스로 문제풀면서 알고리즘 공부 + 추가 지식 보충(언어 고급 활용)

5. 기업별 코테 유형

삼성 : DFS/BFS 집중 IT대기업(네카라쿠배) : 자료구조 등 폭넓은 지식 + 수학적 이론 + String 문자열 활용 중견 스타트업(당근마켓/11번가/ABLY) : 코딩 테스트 안보는 곳도 있고 문제도 쉬운 편 but 실무 질문이 많음

IDE 사용하되 자동완성 막는 경우 대비 : 프로그래머스 같은 웹 환경에서 맨 땅 코딩 연습이 필요

기업 별로 과정, 내용이 상이하기 때문에 후기 열심히 찾아보며 준비하는 것이 바람직하다.
